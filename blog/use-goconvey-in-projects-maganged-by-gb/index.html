Use goconvey in projects maganged by gb
2016-10-19 00:01
1 minute read

這是前陣子在工作上用到的

有個 golang 的專案，使用了 gb 來管理第三方套件

當時想套 goconvey 進來

但是一直搞不定執行的方式

後來在 github goconvey issue 裡找到了方式

$ cd /to/your/gb_projects
$ PROJECT_DIR=`pwd`
$ GOPATH="$PROJECT_DIR/vendor:$PROJECT_DIR" goconvey -packages=1 -port 8899

這樣就會跑 web ui 出來摟

不想跑 web ui 就執行 gb test -v 就可以了…上面這樣只是單純想跑 web ui 而已 XD
