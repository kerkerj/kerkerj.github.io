Playground - Class, Struct

再來就是要提重點的東西了!

class 和 struct 是用來建構整個程式重要的東西

兩者的差異在於：class 可以繼承其他 class，但是 struct 無法繼承

以下是宣告方式

注意，Swift 的 class 和 struct 命名慣例是大寫命名風格 UpperCamelCase

而屬性和方法則是 lowerCamelCase

例如 increaseCount, addTwoInts
