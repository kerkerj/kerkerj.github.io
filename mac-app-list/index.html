Mac app list

下面列了我常用或工作上會用到的軟體，隨時會更新 :)

Tools

  • Alfred3 - 免費

有 powerpack 另外付費，據說非常好用，我目前是暫時拿它來替換掉 Spotlight 而已

看到推友說如果你沒什麼深入在用 spotlight，那就裝了取代他 XD

  • AppCleaner - 刪東西用的

  • CCleaner - 清東西用的

  • DockView - Dock 有 preview 功能

  • Dropbox - 這個之前 mac client … 黑黑的 XD

  • Evernote - 拿來做筆記的，但是現在很少用了…

我還是習慣用 markdown 語法，而看同事都用 Quiver，看起來不錯

只是目前還沒打算花下去 XD (這個看起來似乎也不錯，或許可以試試看 http://writeapp.net/mac/)

  • ForkLift - FTP/SFTP client 同事推薦，但還沒機會用

  • iTerm2 - 我裝 nightly build :P

  • Junos Pulse - VPN

  • Keka - 強大的壓縮/解壓縮軟體

  • Karabiner - A powerful and stable keyboard customizer for OS X.

  • Lastpass - 管密碼用的

  • MemoryClean - 我只拿它來看記憶體用量

  • Mweb Lite - 心智圖 client，之前單純看會用 MindMup，以及 MindNode

  • Seli - Utility for the caps lock key and some international keys in PC keyboards.

  • Spectacle - 視窗整理

  • SSH Tunnel Manager (少用)

  • TunnelBlick - VPN

  • Typora - markdown editor

最早以前用 mou，後來擺爛後改用 MacDown，現在用 Typora

MacDown 我覺得也不錯，就是挑個順手的用。

  • XtraFinder ** - 大推，Finder 的替代品，可以右鍵新增純文字檔案，

在該目錄開啟終端機，以及支援 command - x 剪下功能等等

在 macOS 10.10 後 需要啟用 SIP，而且要設定一些麻煩東西，沒再繼續用了 QQ

Development

  • Android Studio (現在沒在用)

  • Anvil - pow

  • Atom

  • Cakebrew - homebrew GUI, 另外大推 homebrew

  • Dash - 可下載許多程式語言的文件 (付費)

  • Daisy Disk - 分析磁碟用量

  • IntelieJ IDEA CE

  • Insomnia - 另一個 REST API Client，但我還是用 Postman 比較多 XD

  • LiveReload - 程式碼存檔自動重整瀏覽器頁面

  • MySQLWorkbench

  • Postico - Postgres GUI client，免費的就很好用了

功能不多但操作順暢，大多數時候拿來看看資料改改資料就非常夠用了。

  • Postman - 老牌 REST API client，最常用 (command line 會用 httpie)

  • rdm - redis gui client

  • ScreenHero - 分享畫面用，公司用 Slack，所以順便也用了這套，好用

  • Sequel Pro - view sqlite file

  • Shuttle - ssh url short cut

  • SourceTree

  • Xcode

  • eclipse (現在沒在用)

  • SublimeText 3 - one of my favorite text editor, lots of packages

  • TeamViewer

  • Wireshark
