若不小心把 APN 砍掉了可以新增回來(不過通常會有"回復預設值"的選項)

CHT emome

```
1. 名稱：CHT emome
2. APN：emome
```

CHT internet

```
1. 名稱：CHT internet
2. APN：internet
```

CHT mms

```
1. 名稱：CHT mms
2. APN：emome
3. Proxy：10.1.1.1
4. 連線埠：8080
5. MMSC：http://mms.emome.net:8002
6. MMS proxy：10.1.1.1
7. MMSN連線埠：8080
8. MCC：466
9. MNC：92
10. APN類型：mms
```
