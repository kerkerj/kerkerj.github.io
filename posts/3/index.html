請注意，使用系統 Node.js 的 npm 所安裝的套件，是屬於系統 Node.js 的

切換到使用者目錄底下的 Node.js 時，是不會共用系統 npm 的套件的

請注意這一點，接下來我們都會使用 nvm 的 node.js
